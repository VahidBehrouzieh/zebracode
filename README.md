# ZebraCode – Free Online Developer Tools

A collection of 70+ free, client-side developer tools: JSON converters, CSS/SCSS, GraphQL, date/time utilities, encoders, and more. All tools run locally in your browser. No sign-up required.

🔗 **Live Site:** [zebracode.ir](https://zebracode.ir)

## 🛠️ Tools

- **JSON:** to TypeScript, Go, GraphQL, Java, Zod, YAML, etc.
- **CSS:** to JS, SCSS, Tailwind; SCSS to CSS
- **GraphQL:** to TypeScript, Flow, Java, Schema AST, components
- **Date & Time:** Unix timestamp converter, date difference calculator
- **Encoders:** Base64, JWT decoder
- **Others:** HTML to JSX, Markdown to HTML, YAML/TOML/XML conversion, and more.

(For a full list, see [tools.json](./scripts/tools.json) or the home page.)
